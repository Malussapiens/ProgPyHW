Хранение данных:
    - файлы формата csv:
        - список учеников stud.txt
            формат строки: {student_id:фамилия,имя,[subj_id1, subj_id2, ...]}
        - список предметов subj.txt
            формат строки: {наименование_предмета:{student_id:[оценки]}}
Формат записей БД (* - первичный ключ):
    - запись студента: {*student_id:[last_name,name],[subj_list]}
    - запись предмета: {*subj_name:{student_id:[marks_list]}}

Список функций (+ отмечены реализованные):
Инициализация программы:
    - проверить наличие файлов БД
        загрузить файлы, либо создать новые
    - установить счетчик subj_id (0, либо значение из загруженной БД)
    - установить счетчик student_id (0, либо значение из загруженной БД)
    - вывести список студентов на экран +
    - вывести главное меню на экран + 
Функции БД:
    - получить строку из файла
    - преобразовать строку из файла в формат записи БД (студенты, предметы, оценки)
    - заполнить структуры БД (студенты, предметы, оценки)
    - получить список id студентов +
    - добавить студента в список студентов +
    - получить запись из структуры по id:
        ФИО студента +
        список оценок студента по предметам +
    - получить список предметов +
    - добавить предмет в список предметов +

    - преобразовать запись в формат файла
    - сохранить запись в файл
